SpookyVibes API Documentation

What does the SpookyVibes API do?

This REST API is the server-side component of the SpookyVibes web application. The API (still in progress!) will provide users with access to information about different horror movies, directors, and genres. Users will be able to sign up, update their personal information, and create a list of their favorite scary movies.

Endpoints Overview

Request Method URL/Endpoint Response
Return a list of all movies GET /movies JSON object with details for all movies: Title, Description, Director, Genre, Image, Featured
Return data about a single movie by title GET /movies/:title JSON object with single movie details: Title, Description, Director, Genre, Image, Featured
Return data about a genre by name GET /movies/genre/:genreName JSON object with genre name and description
Return data about a director by name GET /movies/director/:directorName JSON object with director details: Name, Bio, Birth Year
Allow new users to register POST /users JSON object with user details: Name, favoriteMovies, id
Allow users to update user info PUT /users/:id JSON object with user details: Name, favoriteMovies, id
Allow users to add movie to their list of favorites POST /users/:id/:movieTitle Text message confirming the movie has been added to favorites
Allow users to add movie to their list of favorites DELETE /users/:id/:movieTitle Text message confirming the movie has been removed from favorites
Allow existing users to deregister DELETE /users/:id Text message confirming the user has been deleted
